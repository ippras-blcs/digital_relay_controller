== Oneshot environment setup

list:: `usbipd list`
bind:: `usbipd bind --busid 4-1`

=== Attach

attach:: `usbipd attach --wsl --busid 4-1`

=== Run

[source,shell]
cargo ramdisk mount
cargo ramdisk mount -t .embuild

[source,shell]
cargo run
